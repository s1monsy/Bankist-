# Bankist (Vanilla JS)

A simple banking dashboard built with **vanilla JavaScript**.  
Log in, view balance and transaction history, transfer money, request a loan, close an account, and sort movements.

> Educational project inspired by the "Bankist" app (Jonas Schmedtmann course).

## Features

- Login with username + PIN
- Current balance
- Transaction history (deposits / withdrawals)
- Money transfers between accounts
- Loan request (basic validation)
- Close account (confirmation by username + PIN)
- Summary: total in / out / interest
- Sort transactions
